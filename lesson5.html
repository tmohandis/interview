10. Перечислить по пять знакомых вам псевдоклассов и псевдостилей с указанием их функционала.

    Псевдоклассы:

  • :hover - используется при наведении на элемент
  • :focus - используется при фокусе на элементе
  • :first-child - первый дочерний элемент
  • :last-child - последний дочерний элемент
  • :right - правый элемент

    Псевдостили

  • :after - используется при добавлении контента до элемента
  • :before - используется при добавлении контента после элемента
